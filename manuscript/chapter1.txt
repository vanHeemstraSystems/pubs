# Chapter 1

## The Overview

The Tacit system consists of the following layers (listed here in no particular order):

* core: business layer
* skin: layout layer
* mask: style layer
* chat: query layer
* bits: data layer
* kick: monitor layer
* reap: report layer
* flow: flow layer
